Streaming FFT/IFFT Demonstration

 

This design demonstrates how the Xilinx FFT block can be used in a data stream with identical input and output rates. In addition, the demo also shows how to incorporate DDS Compiler blocks and complex multipliers as a subsystem to this model. Both Sine and Cosine output signals are used as inputs into the complex multipliers, realized and verified using Simulink, fixed-point simulation.  A Simulink model is provided for your reference only.

 

Using SysGen and ISE 10.1, the following operating frequency is obtained for these two target devices:

-  Virtex-5 xc5vsx50t-1ff1136:  3.236 ns with 5 levels of logic

-  Spartan-3A DSP xc3sd3400a-4fg676: 5.655 ns with 7 levels of logic
