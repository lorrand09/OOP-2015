Release Information : Release Notes 12.4
Release Notes 12.4
System Generator Enhancements
Performance improvements
•
For large designs, simulation and netlisting speed has been improved by 20% over the 12.3 Release.
•
The first-time simulation initialization of models has also been reduced by 15% to 35% compared to the 12.3 Release.
Support for new FIR Compiler 6.1 Block with Fixed Fractional Rate Support
FIR Compiler 6.1 provides Fixed Fractional Rate support so you can specify an integer or a fixed fraction rate for the Rate Change Type.
MATLAB 2010b Production Support
MATLAB 2010b is now supported for production designs.
Updated System Generator Examples with AXI Interface
System Generator examples have been updated to include examples with AXI Interfaces. The examples can be found under the topic AXI4 Conversion Examples.
